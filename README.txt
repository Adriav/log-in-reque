Por problemas tecnicos con la plataforma 'https://www.heroku.com' no se puede mostrar la pagina en internet, asi que se debera trabajar en local

Para hacerlo debe tener instalado el Node JS y npm (ambos se consiguen con el mismo instalador) 
el cual se puede conseguir en el siguiente link: https://nodejs.org/en/

Una vez instalado debe abrir la consola de comandos y ejecutar el comando 'npm -v' para
verificar la version del programa, si en consola sale un numero significa que el programa fue instalado correctamente.
Ahora desde la misma terminal se dirige a la carpeta llamada "node-slq" (Si la descargo de github, la carpeta se llamaria "log-in-reque")
(Contiene los archivos '.env', '.gitignore', 'index.js', 'package.json' y 'package-lock.json')
(Y las carpetas 'controllers', 'node_modules', 'public', 'routes' y 'views')
En esa misma carpeta "node-sql" desde terminal ejecuta el comando 'npm start'
Si todo sale bien los ultimos dos mensajes en consola deberian ser
"Server on port: <puerto>" (tal vez le aparezca el puerto 4000)
y "Conectado!"

Luego de ejecutar el comando (y sin cerrar la terminal), se puede dirigir al browser y escribir en la barra de direcciones:
"localhost:4000" (O el puerto que le haya aparecido en consola)
Y ya esta listo para probar el programa. 
Para finalizar el programa debe presionar 'ctrl'+C en la consola o simplemente cerrar la consola (Puede que tarde un poco mas en detenerse)
En caso de error en consola sale el log.
